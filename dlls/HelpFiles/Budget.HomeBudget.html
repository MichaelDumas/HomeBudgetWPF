GetBudgetDictionaryByCategoryAndMonth(Nullable<DateTime>, Nullable<DateTime>, Boolean, Int32)

Get a list of Dictionaries with information about the total expenses for a given month and category, in addition to the totals for a given month, and the totals for a given category.

Each item in the returned list is a Dictionary (list of string/object pairs), where the string/objects are:

KEYVALUE
MonthA string representing the month
Category Name 1A double representing the total expenditures for this month and category
details:Category Name 1A list of BudgetItems for this month and category
Category Name 2A double representing the total expenditures for this month and category
details:Category Name 2A list of BudgetItems for this month and category
...
Category Name nA double representing the total expenditures for this month and category
details:Category Name nA list of BudgetItems for this month and category
TotalA double representing the total expenditures for this month

The last item in the returned list is a Dictionary (list of string/object pairs), where the string/objects are:

KEYVALUE
Monththe string 'TOTALS'
Category Name 1A double representing the total expenditures for this category
Category Name 2A double representing the total expenditures for this category
......
Category Name nA double representing the total expenditures for this category

Declaration
public List<Dictionary<string, object>> GetBudgetDictionaryByCategoryAndMonth(DateTime? Start, DateTime? End, bool FilterFlag, int CategoryID)
Parameters
Type Name Description
System.Nullable<System.DateTime> Start

Only expenses that were incurred after or on this date will be processed

default: Jan 1, 1900
System.Nullable<System.DateTime> End

Only expenses that were incurred before or on this date will be processed

default: Jan 1, 2500
System.Boolean FilterFlag

If set, only expenses with CategoryId will be processed
System.Int32 CategoryID

If FilterFlag is set, only expenses with this category ID will be processed
Returns
Type Description
System.Collections.Generic.List<System.Collections.Generic.Dictionary<System.String, System.Object>>

A collection of Dictionaries
Remarks

If Start and End date are null, then they will be set to their defaults

Examples
   // open an existing budget
   var myBudget = new HomeBudget("existingDB.db");

   // Get a report of my expenses by month
   int categoryID = 0;
   var report = myBudget.GetBudgetDictionaryByCategoryAndMonth(
       null, null, false,0);

   if (report.Count < 1) return;

   // Print the report
   var totals = report[report.Count - 1];
   String horizontal_line = new string('-', totals.Count * 15 + 20);
   Console.WriteLine(horizontal_line);

   Console.Write(String.Format("{0,-10} |","Month"));
   foreach (var category in myBudget.categories.List())
   {
       var descr = category.Description;
       if (totals.ContainsKey(descr))
       {
           Console.Write(String.Format("{0,15} |", 
               descr.Substring(0, Math.Min(12, descr.Length))));
       }
   }
   Console.WriteLine(String.Format("{0,10}", "TOTAL"));
   Console.WriteLine(horizontal_line);

   int counter = 0;
   foreach (var monthDictionary in report)
   {
       counter++;
       if (counter == report.Count ) Console.WriteLine(horizontal_line);

       Console.Write(String.Format("{0,-10} |", monthDictionary["Month"] ));
       foreach (var category in myBudget.categories.List())
       {
           var descr = category.Description;
           if (totals.ContainsKey(descr))
           {
               string amount = new string(' ', 15);
               if (monthDictionary.ContainsKey(descr))
                   amount = String.Format("{0,15:C}", monthDictionary[descr]);
               Console.Write($"{amount} |");
           }
       }
       if (monthDictionary.ContainsKey("Total"))
       {
           Console.WriteLine(String.Format("{0,15:C}", monthDictionary["Total"]));
       }
       else
       {
           Console.WriteLine();
       }
   }
   Console.WriteLine(horizontal_line);
   Console.WriteLine();

GetBudgetItemsByCategory(Nullable<DateTime>, Nullable<DateTime>, Boolean, Int32)

Get a list of BudgetItemsByCategory, filtered by date, and possibly by category.

Each BudgetItemsByCategory include total expenditures for that category, and a list of individual expenses for that category.

Declaration
public List<BudgetItemsByCategory> GetBudgetItemsByCategory(DateTime? Start, DateTime? End, bool FilterFlag, int CategoryID)
Parameters
Type Name Description
System.Nullable<System.DateTime> Start

Only expenses that were incurred after or on this date will be processed

default: Jan 1, 1900
System.Nullable<System.DateTime> End

Only expenses that were incurred before or on this date will be processed

default: Jan 1, 2500
System.Boolean FilterFlag

If set, only expenses with CategoryId will be processed
System.Int32 CategoryID

If FilterFlag is set, only expenses with this category ID will be processed
Returns
Type Description
System.Collections.Generic.List<BudgetItemsByCategory>

A collection of BudgetItemsByCategory objects
Remarks

Results are ordered by Category Description

If Start and End date are null, then they will be set to their defaults

see BudgetItemsByCategory

Examples
       var myBudget = new HomeBudget("existingDB.db");
       var report = myBudget.GetBudgetItemsByCategory(
           null,                // start date, is null so use default (Jan 1, 1900) 
           null,                // end date, is null so use default (Jan 1, 2025)
           false,               // I do NOT want to filter results on category Id?
           9                    // unused category id
           );

           foreach (var categoryItem in report)
           {
               Console.WriteLine();
               Console.WriteLine(
                   $"{categoryItem.Category}, Total: {categoryItem.Total}");

               foreach (var individualItem in categoryItem.Details)
               {
                   Console.WriteLine(
                       $"{individualItem.Date}     " +
                       $"{individualItem.Amount},\t" +
                       $"{individualItem.ShortDescription}");
               }
           }
